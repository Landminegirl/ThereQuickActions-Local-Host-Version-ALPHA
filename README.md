# ThereQuickActions-Local-Host-Version-ALPHA
alpha build of a local host option so you dont need to download any exes &lt;3

works in any browser!

# INSTALL
Download ThereQuickActionsLocalHost.zip
1. Unzip folder
2. Click CLICK TO LAUNCH.html
3. Copy local file address
4. In the There game chat box type in /local or /web to open the There Browser Tab.
5. Paste that copied address into the Address bar in the There browser.
6. Press Enter. ThereQuickActions will open inside the game browser.
